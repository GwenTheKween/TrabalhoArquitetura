Como compilar:
    Na pasta do simulador, para compilá-lo execute o comando:
        make

Instruções suportadas:
    Instruções tipo R com inteiros: ADD, SUB, MUL, DIV
    Instruções tipo R com decimais: ADD.D, SUB.D, MUL.D, DIV.D
    Instruções tipo I: LW, SW, ADDI, SUBI
    Instruções tipo J não são suportadas
