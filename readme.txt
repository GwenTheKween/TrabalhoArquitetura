Como compilar:
    Na pasta do simulador, para compilar e executá-lo utilize o comando:
        make run

Instruções suportadas:
    Instruções tipo R com inteiros: ADD, SUB, MUL, DIV
    Instruções tipo R com decimais: ADD.D, SUB.D, MUL.D, DIV.D
    Instruções tipo I: LW, SW, ADDI, SUBI
    Instruções tipo J não são suportadas

Os registradores suportados são:
    F0, F1, F2, F3, F4, F5 e F6

Não é possível voltar para um ciclo de clock anterior
Para avançar na execução das instruções utilizar SOMENTE teclas que NÃO contém caracteres especiais, 
como barra de espaço, ENTER ou as setas do teclado
Segurar essas teclas avança múltiplos ciclos 

No arquivo instructions.sbd, cada instrução deve estar em uma linha separada

Para parar a execução, pressione "q"
